Institutions and Bodies of the European Union

About

General

This dataset defines Linked Data compatible URIs for major institutions and bodies of the European Union that can be used as identifiers in EU-related datasets.

This dataset defines Linked Data compatible URIs for major institutions and bodies of the European Union. These URIs, identifying organisations on the European level, can be used as canonical identifiers in datasets.

Publisher

The dataset has been published by Michael Hausenblas and Anja Jentzsch, enabled through funding received from the European Community under the Seventh Framework Programme (FP7), Grant Agreement No. 256975, LOD Around-The-Clock (LATC) Support Action.

Structure and Schema

Formats and Examples

The dataset is available in the following formats:

  • In RDFa via http://institutions.publicdata.eu/.
  • In RDF/XML via http://institutions.publicdata.eu/format/rdf-xml.
  • In Turtle via http://institutions.publicdata.eu/format/ttl.
  • In N-Triples via http://institutions.publicdata.eu/format/nt.
  • In JSON via http://institutions.publicdata.eu/format/json.
  • In CSV via http://institutions.publicdata.eu/format/csv.

An example entity of this dataset is the European Commission with the URI: http://institutions.publicdata.eu/#ec.
